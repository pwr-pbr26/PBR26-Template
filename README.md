# PBR26 - Szablon Projektu AI/ML

Szablon dla zespołów realizujących projekty w ramach grupy kursów Projekt Badawczo-Rozwojowy w Inzynierii Oprogramowania.
Zarządzanie zależnościami: `uv`. Uruchamianie zadań: `just`.

## 🚀 Szybki start
1. Zainstaluj [uv](https://docs.astral.sh/uv/) oraz [just](https://just.systems/).
2. Pobierz repozytorium i wejdź do folderu.
3. Uruchom `just setup` aby przygotować wyizolowane środowisko.
4. Uruchom `just jupyter` aby wystartować notebooki.
